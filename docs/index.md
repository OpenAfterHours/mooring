# Mooring

Git-free [marimo](https://marimo.io) notebook sharing via GitHub — analysts with
only Python 3.12 pull, edit, and push a shared repo's notebooks over the GitHub
REST API, no git required.

Get started with the [user guide](users/index.md), set up a team with the
[admin guide](admins/index.md), or dig into the
[developer guide](developers/index.md).
